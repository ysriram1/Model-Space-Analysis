ModelSpace
==========
(Needs Python 2.7)

To launch ModelSpace prototype demo:
- clone repo
- launch terminal
- cd into the repo
- type "python Server.py userDisfunctionSpace.pickle"
- use any browser and go to http://127.0.0.1:11319/ModelSpace
